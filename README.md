## Quick start :rocket:

Make sure you fulfil #requirements

```shell script
git clone https://github.com/twrdyyy/scala_project awesome_project
cd awesome_project
chmod u+x example
./example
```

## Requirements :mega:
 - Python3+ :snake:
 - Open ai gym 
 ```shell script
pip3 install gym
```
 - Scala
 - Numsca 
 \
Simply add this line to your #sbt build
```scala
libraryDependencies += "be.botkop" %% "numsca" % "0.1.7"
```
